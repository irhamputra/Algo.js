### Getting Started

1. Clone this repo
```
https://github.com/irhamputra/algo-learning.git
cd algo-learning
```

2. Install JEST
```
npm install -g jest
```

3. Start JEST
```
jest {dir}/test.js --watch
example
jest reversestring/test.js --watch
```

4. Start answering the question in index.js
```
function reverse(str){
// Tulis jawaban kamu disini

}
```
